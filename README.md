# 大富翁服务端

## 项目简介

这是一个大富翁游戏项目的后端

前端项目地址：<https://github.com/WhellingQ/richman>

demo体验地址：http://richman.qiuluo.xin



项目利用websocket实现多人实时在线游戏。可以注册、登录，创建房间，实时游戏。
